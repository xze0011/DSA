# 977. Squares of a Sorted Array (有序数组的平方)

```java
public int[] sortedSquares(int[] nums) {
    int right = nums.length - 1;
    int left = 0;
    int ep = nums.length - 1; // 从最后面往前面加
    int[] arr = new int[right + 1];
    while (left <= right) {
        // 正数的相对位置是不变的， 需要调整的是负数平方后的位置
        if (nums[left] * nums[left] > nums[right] * nums[right]) {
            arr[ep] = nums[left] * nums[left];
            ep--;
            left++;
        } else {
            arr[ep] = nums[right] * nums[right];
            ep--;
            right--;
        }
    }
    return arr;
}
```
