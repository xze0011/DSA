# 27. Remove Element (移除元素)

## 暴力法 (Brute Force)

```java
public int removeElement(int[] nums, int val) {
    int newl = nums.length;
    for (int i = 0; i < nums.length; i++) {
        if (nums[i] == val) {
            for (int j = i + 1; j < nums.length; j++) {
                nums[j - 1] = nums[j];
            }
            // 后面的 index 往前移了，所以 i--
            i--;
            newl--;
        }
    }
    return newl;
}
```

## 双指针 (Two Pointers)

```java
public int removeElement(int[] nums, int val) {
    int slow = 0;
    for (int i = 0; i < nums.length; i++) {
        // 只接受不删的数组元素
        if (nums[i] != val) {
            nums[slow] = nums[i];
            slow++;
        }
    }
    return slow;
}
```
