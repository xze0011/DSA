# 209. Minimum Size Subarray Sumy (滑动窗口)

滑动窗口的本质是双指针，

1. 窗口内即 元素之和大于等于 target  
2. 如果窗口内值一旦大于 target，就把 left 往前移动以缩小窗口。  
3. 默认 right++ 推动窗口  
```java
public int minSubArrayLen(int target, int[] nums) {
    int res = Integer.MAX_VALUE;
    int sum = 0;
    int left = 0;

    for (int right = 0; right < nums.length; right++) {
        sum += nums[right];
        while (sum >= target) {
            res = Math.min(res, right - left + 1);
            sum -= nums[left];
            // 如果窗口内值一旦大于 target，就把 left 往前移动以缩小窗口。
            left++;
        }
    }

    return res == Integer.MAX_VALUE ? 0 : res;
}
```
