# 303. Range Sum Query - Immutable (区间和)

```java
class NumArray {
    int[] prefix;
    
    public NumArray(int[] nums) {
        int n = nums.length;
        prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + nums[i];
        }
    }
    
    public int sumRange(int left, int right) {
        // 子区间的和
        return prefix[right + 1] - prefix[left];
    }
}
```
