# 144. Binary Tree inorder Traversal

```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;


        while (cur != null || !stack.isEmpty()) {

            // ⭐ 一路向左，把当前子树的“左链”全部入栈

            while (cur != null) {
                stack.push(cur);
                cur = cur.left;
            }

            // ⭐ 左子树走到尽头，开始回溯
            TreeNode node = stack.pop();

            result.add(node.val);

            // ⭐ 转向右子树
            cur = node.right;
        }

        return result;
    }
}
```
