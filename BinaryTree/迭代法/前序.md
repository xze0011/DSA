# 144. Binary Tree Preorder Traversal

```java
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        if (root == null)
            return result;
        stack.push(root);

        while (!stack.isEmpty()) {
            TreeNode mid = stack.pop();
            // 前序遍历：先处理根
            result.add(mid.val);

            // 先右后左
            if (mid.right != null)
                stack.push(mid.right);
            if (mid.left != null)
                stack.push(mid.left);
        }

        return result;
    }
```
