# 700. Search in a Binary Search Tree

```java
    private TreeNode res;
    public TreeNode searchBST(TreeNode root, int val) {
        res = null;
        search(root,val);
        return res;
    }

    private void search(TreeNode root, int val){
        if(root == null) return;

        if(root.val == val) {
            res = root;
            return;
        }
        search(root.left,val);
        search(root.right,val);
    }
```
