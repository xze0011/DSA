# 104. Maximum Depth of Binary Tree

```java
private int max = 0;

public int maxDepth(TreeNode root) {
    int cur = 0;
    dfs(root, cur);
    return max;
}

private void dfs(TreeNode root, int cur) {
    if (root == null) {
        return;
    }

    cur = cur + 1;
    dfs(root.left, cur);
    dfs(root.right, cur);

    max = Math.max(max, cur);
}
```
