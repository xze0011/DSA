# 226. Invert Binary Tree

只有前序和后序可以使用，中序会打乱遍历顺序。

```java
public TreeNode invertTree(TreeNode root) {
    dfs(root);
    return root;
}

private void dfs(TreeNode root) {
    if (root == null) return;

    // 前序直接换并不会破坏往下走的顺序
    TreeNode temp = root.left;
    root.left = root.right;
    root.right = temp;

    dfs(root.left);
    dfs(root.right);
}
```
