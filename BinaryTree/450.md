# 450. Delete Node in a BST

```java
public TreeNode deleteNode(TreeNode root, int key) {
    // 基本情况：如果树为空，返回 null
    if (root == null) return root;

    // 第一步：遍历树以找到要删除的节点
    if (root.val > key) {
        root.left = deleteNode(root.left, key);
    } else if (root.val < key) {
        root.right = deleteNode(root.right, key);
    } else {
        // 第二步：找到要删除的节点

        if (root.left == null && root.right == null) {
            // 情况 1：节点是叶子节点（没有子节点）
            root = null;
        } else if (root.left == null) {
            // 情况 2：节点只有右子节点
            root = root.right; // 用右子节点替换当前节点
        } else if (root.right == null) {
            // 情况 3：节点只有左子节点
            root = root.left; // 用左子节点替换当前节点
        } else {

            // 情况 4：节点有两个子节点
            // 找到中序后继（右子树中最小的节点）
            TreeNode successor = findMin(root.right);
            // 用后继节点的值替换当前节点的值 ， 为了避免破坏结构
            root.val = successor.val;
            // 从右子树中删除后继节点
            root.right = deleteNode(root.right, successor.val);
        }
    }
    return root;
}

// 辅助函数：在子树中找到最小值节点
private TreeNode findMin(TreeNode node) {
    while (node.left != null) {
        node = node.left; // 不断向左遍历以找到最小值
    }
    return node;
}
```

### 关键步骤解释:
1. **基本情况**: 如果树为空（`root == null`），返回 `null`。
2. **遍历**: 根据 `key` 是小于还是大于当前节点的值，递归查找左子树或右子树。
3. **节点删除**:
   - 如果节点是叶子节点，直接将其设置为 `null`。
   - 如果节点只有一个子节点，用其子节点替换。
   - 如果节点有两个子节点，找到中序后继，用后继的值替换当前节点的值，并删除后继。
4. **找到中序后继**: 中序后继是右子树中最小的节点，通过不断向左遍历直到 `node.left == null` 找到。

这种方法确保删除后仍然保持 BST 的性质。
