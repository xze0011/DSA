# 106. Construct Binary Tree from Inorder and Postorder Traversal

```java
HashMap<Integer, Integer> map;
int postorderlast = 0;

public TreeNode buildTree(int[] inorder, int[] postorder) {
    if (inorder == null || postorder == null || inorder.length == 0) {
        return null;
    }
    TreeNode root = new TreeNode();
    postorderlast = postorder.length - 1;
    map = new HashMap<>();
    for (int i = 0; i < inorder.length; i++) {
        map.put(inorder[i], i);
    }
    return dfs(inorder, postorder, 0, postorder.length - 1);
}

private TreeNode dfs(int[] inorder, int[] postorder, int left, int right) {
    if (left > right) return null;
    int rootVal = postorder[postorderlast--];

    // 实际上赋值的地方，Inorder 只提供两侧的位置
    TreeNode root = new TreeNode(rootVal);
    int mid = map.get(rootVal);

    // ⚠ 注意顺序：必须先构造右子树，再构造左子树！
    // 因为 postorder（后序）顺序是： left → right → root
    // 从后往前取时顺序变成：root → right → left
    // 所以右子树先被处理，再处理左子树
    root.right = dfs(inorder, postorder, mid + 1, right);
    root.left = dfs(inorder, postorder, left, mid - 1);
    return root;
}
```
