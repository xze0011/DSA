# 538. Convert BST to Greater Tree

```java
    int cur = 0;
    public TreeNode convertBST(TreeNode root) {
        dfs(root);
        return root;
        
    }
    private void dfs(TreeNode root){
        if(root == null) return;


// 中序遍历反过来，先到最右侧
        dfs(root.right);
        cur = cur + root.val;
        root.val = cur;
        dfs(root.left);
 
    }
```
