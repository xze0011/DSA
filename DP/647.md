# 647. Palindromic Substrings

```java
        int n = s.length();
        boolean[][] dp = new boolean[n][n];
        //初始化默认false
        int res = 0;
        for (int i = n - 1; i >= 0; i--) {
            for (int j = i; j < n; j++) {
                if (s.charAt(i) == s.charAt(j)) {
                    if (j - i <= 1) {
                        dp[i][j] = true;
                    } else {
                        // 遍历顺序由这个决定，dp[i+1]必须先于[i]被计算
                        dp[i][j] = dp[i + 1][j - 1];
                    }
                    if (dp[i][j])
                        res++;
                }

            }
        }
        return res;
```
