# 516. Longest Palindromic Subsequence

```java
    public int longestPalindromeSubseq(String s) {
        int n = s.length();
        if (n == 1)
            return 1;
        // Longest Palindromic Subsequence of [i,j]
        int[][] dp = new int[n][n];
        int res = 0;
        for (int i = n - 1; i >= 0; i--) {
            for (int j = i; j < n; j++) {
                if (s.charAt(i) == s.charAt(j)) {
                    if (j == i) {
                        dp[i][j] = 1;
                    } else if (j - i == 1) {
                        dp[i][j] = 2;
                    } else {

                        dp[i][j] = dp[i + 1][j - 1] + 2;
                    }
                } else {
                    // 选一边[i , j-1] or [i+1,j]
                    dp[i][j] = Math.max(dp[i][j - 1], dp[i + 1][j]);
                }
                res = Math.max(dp[i][j], res);
            }
        }
        return res;
```
