# 674. Longest Continuous Increasing Subsequence

```java
    public int findLengthOfLCIS(int[] nums) {
        int n = nums.length;
        int[] dp = new int[n];
        if (n == 1)
            return 1;
        Arrays.fill(dp, 1);
        int res = 0;
        for (int i = 1; i < n; i++) {
            if (nums[i] > nums[i - 1]) {
                dp[i] = dp[i - 1] + 1;
            }
            res = Math.max(res, dp[i]);

        }
        return res;
    }
```
