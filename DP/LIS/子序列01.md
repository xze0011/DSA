# 300. Longest Increasing Subsequence

```java
    public boolean isSubsequence(String s, String t) {
        int ns = s.length();
        int nt = t.length();

// 在 t 的前 j 个字符（t[0..j-1]）中，
// 最多能按顺序匹配到 s 的前 i 个字符（s[0..i-1]）中的字符数量
        int[][] dp = new int[ns+1][nt+1];

        // base case：
        // dp[0][j] = 0：空串 s 的前 0 个字符，永远能匹配 0 个
        // dp[i][0] = 0：t 的前 0 个字符，最多也只能匹配 0 个（Java 默认 0，所以不必显式写）

        for(int i = 1; i < ns+1; i++){
            for(int j = 1; j < nt+1; j++){
                if(s.charAt(i-1) == t.charAt(j-1)){
                    dp[i][j] = dp[i-1][j-1] + 1;
                }else{
                    dp[i][j] = dp[i][j-1];
                }

            }
        }
        return dp[ns][nt] == s.length();
    }
```
