# 746. Min Cost Climbing Stairs

```java
int n = cost.length;
        //到达第 i 个台阶所需要的最小花费
int[] dp = new int[cost.length];
dp[0] = cost[0];
dp[1] = cost[1];
for (int i = 2; i < cost.length; i++) {
    dp[i] = Math.min(dp[i - 1], dp[i - 2]) + cost[i];
}
        // 到达楼顶，可以从最后一步 or 倒数第二步跨上去
return Math.min(dp[n - 1], dp[n - 2]);
```
