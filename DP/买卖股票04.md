# 123. Best Time to Buy and Sell Stock III

```java
    public int maxProfit(int k, int[] prices) {
        int n = prices.length;
        int[][] dp = new int[n][2 * k];

        for (int i = 0; i < 2 * k; i++) {
            if (i % 2 == 0) {
                dp[0][i] = -prices[0];
            }
        }

        // dp[i][0] 表示：在第 i 天结束时，
        //                第一次手里【持有一支股票】的情况下，
        //                能获得的【最大利润】
        //
        // dp[i][1] 表示：在第 i 天结束时，
        //                第一次手里【不持有股票】的情况下，
        //                能获得的【最大利润】

        // dp[i][2] 表示：在第 i 天结束时，
        //                第二次手里【持有一支股票】的情况下，
        //                能获得的【最大利润】

        // dp[i][3] 表示：在第 i 天结束时，
        //                第二次手里【不持有一支股票】的情况下，
        //                能获得的【最大利润】

        for (int i = 1; i < n; i++) {
            for (int j = 0; j < 2 * k; j = j + 2) {
                if (j == 0) {
                    // 第一次情况 0 - prices[i]
                    dp[i][0] = Math.max(dp[i - 1][0], -prices[i]);
                    dp[i][1] = Math.max(dp[i - 1][1], dp[i - 1][0] + prices[i]);
                } else {
                    // 第二次之后的情况
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - 1] - prices[i]);
                    dp[i][j + 1] = Math.max(dp[i - 1][j + 1], dp[i - 1][j] + prices[i]);
                }
            }

        }
        //dp[n-1][3] ≥ dp[n-1][1]
        return dp[n - 1][2 * k - 1];
    }
```
