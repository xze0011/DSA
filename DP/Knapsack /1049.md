# 1049. Last Stone Weight II

```java
    public int lastStoneWeightII(int[] stones) {
        int sum = 0;
        for(int num: stones){
            sum += num;
        }
        int target = sum / 2;
        
//在“容量”为 j 的情况下，最多能凑出的石头总重量
        int[] dp = new int[target + 1];

        for(int i = 0; i < stones.length; i++){
                  int currentVal = stones[i]; // 当前石头的重量
            for(int j = target; j >= currentVal; j--){
                dp[j] = Math.max(dp[j],dp[j-currentVal] + currentVal);
            }
        }

        return sum - dp[target] * 2;
    }
```
