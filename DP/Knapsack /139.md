# 139. Word Break

```java
    public boolean wordBreak(String s, List<String> wordDict) {
        HashSet<String> set = new HashSet<>(wordDict);
        int n = s.length();
        boolean[] dp = new boolean[n + 1];
        dp[0] = true;
        for (int j = 0; j < n + 1; j++) {
            for (String w : wordDict) {
                int L = w.length();
                if(j >=L){
                    // 存在任意一个单词 w”能让 j 成立，dp[j] 就应该是 true
                    dp[j] = dp[j] || (dp[j-L] && s.substring(j-L,j).equals(w));
                }
            }
        }

        return dp[n];
    }
```
