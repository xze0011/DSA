# 322. Coin Change

```java
    public int coinChange(int[] coins, int amount) {
        int n = coins.length;

        // the fewest number of coins that you need to make up that amount j
        int[] dp = new int[amount+1];
        Arrays.fill(dp, Integer.MAX_VALUE);
        dp[0] = 0;

        for(int j = 1; j < amount + 1; j++){
            for(int i = 0; i < n; i++){
                int currentAmount = coins[i];
                if(j >= currentAmount && dp[j - currentAmount] != Integer.MAX_VALUE){
                    // Integer.MAX_VALUE + 1 就会溢出
                    dp[j] = Math.min(dp[j],dp[j-currentAmount]+1);
                }
            }
        }

    // 如果最终仍然是不可达状态，返回 -1
        return dp[amount] == Integer.MAX_VALUE ? -1 : dp[amount];
    }
```
