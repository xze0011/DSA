# 279. Perfect Squares

与 322 高度相似

```java
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        Arrays.fill(dp, Integer.MAX_VALUE);
        dp[0] = 0;

        for (int j = 1; j < n + 1; j++) {
            // 这里直接用 平方
            for (int i = 1; i * i <= n; i++) {
                int sq = i * i;

                if (j >= sq && dp[j - sq] != Integer.MAX_VALUE) {
                    dp[j] = Math.min(dp[j], dp[j - sq] + 1);
                }
            }
        }

        return dp[n];
    }
```
