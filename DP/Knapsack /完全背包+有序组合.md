# 377. Combination Sum IV

完全背包

```java
    public int combinationSum4(int[] nums, int target) {
        int n = nums.length;
        int[] dp = new int[target + 1];
        dp[0] = 1;

        for (int j = 1; j < target + 1; j++) {
            for (int i = 0; i < n; i++) {
                int currentAmount = nums[i];
                if (j >= currentAmount) {
                    dp[j] = dp[j] + dp[j - currentAmount];
                }

            }
        }
        return dp[target];
    }
```
