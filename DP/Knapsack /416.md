# 416. Partition Equal Subset Sum

```java
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for (int num : nums) {
            sum += num;
        }

        // 如果总和是奇数，不能平分
        if (sum % 2 != 0) {
            return false;
        }

        int target = sum / 2;

        // dp[j] 表示：容量为 j 时，能凑到的最大和
        int[] dp = new int[target+1];

      for (int i = 0; i < nums.length; i++) {
            int  currentVal = nums[i];
           for (int j = target; j >= currentVal; j--){
                int hasi = dp[j];

               
                int donthasi = dp[j- currentVal] +  currentVal;
                dp[j] = Math.max(hasi,donthasi);
            }
        }

        return dp[target] == target;
    }
```
