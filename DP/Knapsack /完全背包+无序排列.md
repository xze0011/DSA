# 518. Coin Change II

完全背包

```java
    public int change(int amount, int[] coins) {
        int n = coins.length;

        int[] dp = new int[amount+1];
        dp[0] = 1;

        for(int i = 0; i < n; i ++){
            int currentAmount = coins[i];

            for(int j = currentAmount; j < amount+1 ; j++){
                dp[j] = dp[j] + dp[j - currentAmount];
            }
        }

        return dp[amount];
    }
```
