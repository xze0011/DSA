# 70. Climbing Stairs

```java
    public int climbStairs(int n) {
        int[] dp = new int[n+1];
        dp[0] = 1;
        dp[1] = 1;
        dp[2] = 2;

        for(int i = 3; i < n+1;i++){
          // 前一个台阶和前前一个台阶的组合总数
            dp[i] = dp[i-1] + dp[i-2];
        }

        return dp[n];
    }
```
