# 541. Reverse String II

```java
public String reverseStr(String s, int k) {
    char[] sa = s.toCharArray();

    for (int n = 0; n < sa.length; n += 2 * k) {

        // 处理不足 2 * k的情况
        int right = Math.min(n + k - 1, sa.length - 1);
        for (int left = n; left < right; left++) {
            char temp = sa[left];
            sa[left] = sa[right];
            sa[right] = temp;
            right--;
        }
    }
    // 主要接受 char[]， String, StringBuilder
    return new String(sa);
}
```
