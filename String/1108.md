# 1108. Defanging an IP Address

### StringBuilder 操作方法

```java
public String defangIPaddr(String address) {
    StringBuilder sb = new StringBuilder();
    
    for (char c : address.toCharArray()) {
        if (Character.isDigit(c)) {
            sb.append(c);
        } else {
            sb.append("[.]");
        }
    }
    
    return sb.toString();
}
```
