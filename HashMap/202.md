# 202. Happy Number

```java
public boolean isHappy(int n) {
    Set<Integer> set = new HashSet<>();
    while (n != 1 && !set.contains(n)) {
        set.add(n);
        n = getNext(n);
    }
    return n == 1;
}

private int getNext(int n) {
    int res = 0;
    while (n > 0) {
        // Extract the last digit
        int temp = n % 10;
        res += temp * temp;

        // Remove the last digit
        n = n / 10;
    }
    return res;
}
```
