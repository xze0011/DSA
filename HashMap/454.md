# 454. 4Sum II

```java
public int fourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
    Map<Integer, Integer> map = new HashMap<>();
    int res = 0;
    // 如何分组比如 [nums1, nums3]结果不变
    for (int i : nums1) {
        for (int j : nums2) {
            int sum = i + j;
            map.put(sum, map.getOrDefault(sum, 0) + 1);
        }
    }
    for (int i : nums3) {
        for (int j : nums4) {
            int sum = i + j;
            // 当 key 不存在时，不做任何贡献、相当于忽略掉。
            res += map.getOrDefault(-sum, 0);
        }
    }
    return res;
}
```
