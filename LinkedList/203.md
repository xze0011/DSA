# 203. Remove Linked List Elements (移除链表)

## 不用虚拟节点

```java
public ListNode removeElements(ListNode head, int val) {
    // 先单独处理空的头，或者是要删除头的情况
    while (head != null && head.val == val) {
        head = head.next;
    }

    if (head == null) return head;
    ListNode prev = head;
    ListNode cur = head.next;
    while (cur != null) {
        if (cur.val == val) {
            prev.next = cur.next;
        } else {
            prev = cur;
        }
        cur = cur.next;
    }
    return head;
}
```

## 使用虚拟节点

```java
public ListNode removeElements(ListNode head, int val) {
    ListNode dummy = new ListNode(0, head);
    ListNode cur = dummy;

    while (cur.next != null) {
        if (cur.next.val == val) {
            cur.next = cur.next.next;
        } else {
            cur = cur.next;
        }
    }
    return dummy.next;
}
```
