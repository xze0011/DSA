# 206. Reverse Linked List (反转链表)

通过 `last` 逐步更改，通过 `temp` 暂存断开的下一个目标。

```java
public ListNode reverseList(ListNode head) {
    ListNode temp = null;
    ListNode cur = head;
    ListNode last = null;
    while (cur != null) {
        temp = cur.next;
        cur.next = last;
        last = cur;
        cur = temp;
    }
    return last;
}
```
