# 198. House Robber

```java
    public int rob(int[] nums) {
        int n = nums.length;

        //dp[i] = 抢到第 i 间房子（包括下标 i）为止，能得到的最大金额
        int[] dp = new int[n];

        dp[0] = nums[0];
        dp[1] = Math.max(nums[0],nums[1]);

        for(int i = 2; i < n; i++){
            // // 不抢第 i 间  ｜ 抢第 i 间
            dp[i] = Math.max(dp[i - 1],  dp[i-2] +nums[i]);
        }

        return dp[n-1];

    }
```
