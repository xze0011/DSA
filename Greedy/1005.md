# 1005. Maximize Sum Of Array After K Negations

```java
class Solution {
    public int largestSumAfterKNegations(int[] nums, int k) {
        Arrays.sort(nums);

        int sum = 0;
        int minAbs = Integer.MAX_VALUE;

        for (int i = 0; i < nums.length; i++) {
            // 优先把负数翻成正数
            if (nums[i] < 0 && k > 0) {
                nums[i] = -nums[i];
                k--;
            }

            sum += nums[i];
            minAbs = Math.min(minAbs, nums[i]); // 记录最小绝对值
        }

        // 如果 k 还剩奇数次，翻最小的数一次，
        if (k % 2 == 1) {
            sum -= 2 * minAbs; // 扣除且减去
        }

        // 如果是偶数次就不用考虑

        return sum;
    }
}
```
