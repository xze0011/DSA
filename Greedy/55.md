# 53. Maximum Subarray

```java
   public boolean canJump(int[] nums) {

        int maxJump = 0;
        for(int i = 0; i < nums.length; i++){
            // 已经无法前进
            if (i > maxJump) {
                return false;
            }
            maxJump = Math.max(maxJump, i + nums[i]);
            if(maxJump >= nums.length - 1) {
                return true;
            }
        }
        return false;
    }
```
