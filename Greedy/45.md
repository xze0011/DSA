# 45. Jump Game II

```java
    public int jump(int[] nums) {
        int count = 0;
        int maxJump = 0;   // 当前跳范围内，下一跳能到的最远位置
        int curEdge = 0;   // 当前这一跳的最远边界

        for (int i = 0; i < nums.length - 1; i++) {
            maxJump = Math.max(maxJump, i + nums[i]);

            // 走到当前跳的边界，说明这一跳用完了，必须跳一次
            if (i == curEdge) {
                count++;
                curEdge = maxJump; // 更新为下一跳的覆盖边界
            }
        }
        return count;
    }
```
