# 135. Candy

```java
public int candy(int[] ratings) {
    int n = ratings.length;
    if (n == 1) return 1;

    int[] candy = new int[n];

    Arrays.fill(candy, 1);

    // 1 从左往右遍历：处理「上坡」
    // 如果右边评分更高，糖果必须比左边多
    for (int i = 1; i < n; i++) {
        if (ratings[i] > ratings[i - 1]) {
            candy[i] = candy[i - 1] + 1;
        }
    }

    // 2 从右往左遍历：处理「下坡」
    // 如果左边评分更高，糖果必须比右边多
    // Math.max防止破坏前一次（左→右）已经满足的约束
    for (int i = n - 2; i >= 0; i--) {
        if (ratings[i] > ratings[i + 1]) {
            candy[i] = Math.max(candy[i], candy[i + 1] + 1);
        }
    }

    int sum = 0;
    for (int c : candy) sum += c;
    return sum;
}
```
