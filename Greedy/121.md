# 121. Best Time to Buy and Sell Stock

取最左最小值，取最右最大值，那么得到的差值就是最大利润。
```java
    public int maxProfit(int[] prices) {
        int n = prices.length;
        int min = Integer.MAX_VALUE;
        int res = 0;
        for(int i = 0; i < n; i++){
            min = Math.min(min,prices[i]);
            res = Math.max(res,prices[i]-min);
        }

        return res;
    }
```
