# 455. Assign Cookies
```java
    public int findContentChildren(int[] g, int[] s) {
        Arrays.sort(g);
        Arrays.sort(s);
        int count = 0;
        int cookieStart = s.length - 1;

        for (int i = g.length - 1; i >= 0&& cookieStart>=0; i--) {
            if (g[i] <= s[cookieStart]) {
                count++;
                cookieStart--;
            } 
        }
        return count;
    }
```
