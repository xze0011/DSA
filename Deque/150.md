# 150. Evaluate Reverse Polish Notation

RPN 规则：遇到运算符时要弹出两个数

```java
    public int evalRPN(String[] tokens) {
        Deque<Integer> stack = new ArrayDeque<>();
        for (String c : tokens) {
            if (!c.equals("+") && !c.equals("-") && !c.equals("*") && !c.equals("/")) {
                stack.push(Integer.parseInt(c));
            } else if (c.equals("+")) {
                stack.push(stack.pop() + stack.pop());
            } else if (c.equals("*")) {
                stack.push(stack.pop() * stack.pop());
            } else if (c.equals("-")) {
                //先弹出的为右操作数，后弹出的为左操作数
                int b = stack.pop();
                int a = stack.pop();
                stack.push(a - b);
            } else if (c.equals("/")) {
                int b = stack.pop();
                int a = stack.pop();
                stack.push(a / b);
            }
        }

        return stack.pop();
    }
```
