# 1047. Remove All Adjacent Duplicates In String

```java
public String removeDuplicates(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (stack.isEmpty() || stack.peek() != c) {
            stack.push(c);
        } else {
            stack.pop();
        }
    }
    // 否则需要反转stack
    String str = "";
    while (!stack.isEmpty()) {
        str = stack.pop() + str;
    }
    return str;
}
```
