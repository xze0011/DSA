# 225. Implement Stack using Queues

The default behavior of a Deque is similar to a stack:
```java
class MyStack {
    Deque<Integer> dq;

    public MyStack() {
        dq = new ArrayDeque<>();
    }
    
    public void push(int x) {
        dq.push(x); // Alternatively, dq.addLast(x)
    }
    
    public int pop() {
        return dq.pop(); // Alternatively, dq.removeLast()
    }
    
    public int top() {
        return dq.peek(); // Alternatively, dq.peekLast()
    }
    
    public boolean empty() {
        return dq.isEmpty();
    }
}
```
